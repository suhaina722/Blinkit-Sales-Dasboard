# Blinkit-Sales-Dashboard
Project Overview
The Blinkit Sales & Customer Insights Dashboard is designed to analyze Blinkit’s sales performance, customer satisfaction, and inventory distribution. This project provides actionable insights to support data-driven decision-making and identify opportunities for business optimization using Power BI.

Business Objective
To conduct a comprehensive analysis of Blinkit’s:
Sales Performance
Customer Satisfaction
Inventory Distribution

KPI Definitions
Total Sales
Total revenue generated from all items sold.
Average Sales
Average revenue per transaction.
Number of Items
Total number of items sold.
Average Rating
Average customer satisfaction rating


Steps Performed (Cleaning → Dashboard)

Data Cleaning (Power Query)
Handled missing and null values.
Standardized categories (e.g., Low Fat, Regular).
Corrected incorrect data formats (dates and numerical values).
Removed inconsistencies and duplicates.
Created new calculated columns such as Outlet Age.
Ensured the dataset was clean and analysis-ready

Data modeling using DAX
Dashboard creation in Power BI

Key Insights
Medium-sized outlets generate the highest sales.
Tier 3 locations outperform other regions in revenue.
Regular fat products contribute more to total sales than low-fat items.
Fruits, Snack Foods, and Household items are top-performing categories.
Older outlets tend to generate higher sales due to established customer trust.

Tool Used
Power BI -Data visualization and dashboard creation
Power Query -Data cleaning and transformation
DAX (Data Analysis Expressions) -KPI calculations and data modeling
